# 斗牛游戏

一个基于控制台的斗牛扑克游戏实现。

## 游戏规则

1. **游戏人数**：2-6人
2. **使用牌型**：一副标准52张扑克牌（不含大小王）
3. **牌型大小**：K>Q>J>10>9>8>7>6>5>4>3>2>A

## 牛的大小等级

1. **五小牛**：5张牌点数之和≤10（A按1点计算）
2. **五花牛**：5张牌全为J、Q、K
3. **四炸**：5张牌中有4张点数相同
4. **牛牛**：5张牌中3张点数之和为10的倍数，剩余2张点数之和也为10的倍数
5. **牛九**：剩余2张牌点数之和个位为9
6. **牛八**：剩余2张牌点数之和个位为8
7. **...**：以此类推
8. **牛一**：剩余2张牌点数之和个位为1
9. **无牛**：无法选出3张牌点数之和为10的倍数

## 运行游戏

1. 确保您的系统已安装Python 3
2. 打开命令提示符或PowerShell
3. 导航到游戏目录：`cd c:\code\niu_niu_console`
4. 运行游戏：`python niu_niu.py`

## 游戏流程

1. 输入玩家数量（2-6人）
2. 为每个玩家输入名字（第一个玩家为庄家）
3. 非庄家玩家下注（1-100筹码）
4. 系统发牌（每人5张）
5. 系统自动计算牛型
6. 展示配牛结果
7. 庄家与闲家比牌
8. 显示输赢结果
9. 询问是否继续游戏

## 注意事项

- J、Q、K按10点计算
- A按1点计算
- 点数之和超过10的部分取个位（例如13点视为3点）
- 游戏使用中文界面
